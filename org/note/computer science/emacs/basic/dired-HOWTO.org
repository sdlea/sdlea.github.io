|----------+-----------+--------------------------------------------------------|
| C-x d    | M-x dired | 进入 dired 模式                                        |

|          | *[R]*       | 重命名                                                 |
|          | *[D]*       | 删除                                                   |
|          | *[+]*       | 新增文件夹                                             |
